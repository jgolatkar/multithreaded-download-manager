# Java Multi-Threaded Download Manager

A simple Java-based multithreaded download manager with a Swing user interface.  
This project demonstrates the use of Java concurrency, thread coordination, and basic desktop UI development.

## Features

- Multithreaded file downloads
- Pause and resume support
- Download status tracking
- Swing-based graphical interface

## Tech Stack

- Java
- Java Concurrency (Threads)
- Swing (GUI)

## Purpose

This project is built to learn and demonstrate:
- Multithreading in Java
- Safe pause/resume mechanisms
- Separation of GUI and download logic
